Gameboy VST plugin

I’ve been working on an open-source DAW plugin written with JUCE that exposes the sounds of the Nintendo Gameboy (and Color) as a software instrument for making chiptune sounds. It’s a faithful recreation of both the sound and the limitations of the console, because it wraps Shay Green’S Gb_Snd_Emu, the audio implementation used by many emulators including Visual Boy Advance - M.
